
#### Class selector
```css
/* any span element with class 'highlight' */
span.highlight {}


/* select anything that has class 'notebox' and class 'warning' 
e.g. <div class="notebox warning"> */
.notebox.warning {
    border-color: orange,
    font-weight: bold
}
```

#### ID selector
```css
/* find h1, select element whose id is 'heading' */
h1#heading {}
```
